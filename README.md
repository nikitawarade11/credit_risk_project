# 💳 Credit Risk Modeling

## 🔍 Overview
This project predicts the creditworthiness of loan applicants using machine learning techniques. It classifies whether a customer is high-risk or low-risk, helping in better financial decision-making.

---

## 🚀 Features
- Data preprocessing and cleaning  
- Handles imbalanced dataset using SMOTE  
- Builds classification model (Logistic Regression)  
- Evaluates model performance  

---

## 🛠️ Tech Stack
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Imbalanced-learn (SMOTE)  

---

## ⚙️ Workflow
1. Load loan dataset  
2. Clean missing values  
3. Apply SMOTE for balancing data  
4. Train machine learning model  
5. Evaluate model accuracy  

---
